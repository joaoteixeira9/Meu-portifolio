Projeto Integrador: SC Cortes

O sistema possui duas áreas: Admin e Usuário. O Admin cadastra serviços e produtos, gerencia usuários com criação automatizada no banco de dados e arquivos de acordo com seu ID, visualiza agendas por ID, registra vendas e lista funcionários. O Usuário visualiza horários disponíveis, faz agendamentos e consulta produtos. O backend usa PHP e o frontend utiliza Bootstrap para uma interface responsiva. Dados são armazenados em MySQL.
